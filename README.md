# Commander Gold Theme

A Zed theme extension inspired by Norton Commander, Midnight Commander, and classic Borland editors.

Core palette:

- Deep DOS blue background
- Gold primary text
- Cyan, green, red, and white accents for classic terminal-era contrast

## Install locally in Zed

1. Open the command palette.
2. Run `zed: install dev extension`.
3. Select this directory.
4. Choose the `Commander Gold` theme in Zed appearance settings.

## Notes

- Repository: `https://github.com/koctep/z-commander-gold-theme`
- The extension id already uses the recommended `-theme` suffix for Zed theme extensions.
